About the Randomizer:

Currently randomized:

  • Chests & Stickers
  • Starting Weapons
More functionality will be added over time as this randomizer gets developed. Furthermore it's
not guaranteed that you will be able to find everything like All Stickers etc. This will be changed
in the future.

How to get started:

First of head to the discord (Link at the bottom) and downloaded the latest version of Sonic's Open
World lua file as well as download a seed with the button at the top, then follow either of the
two guides:

  • LuaFrontend

You want to download LuaFrontend and unpack the .zip folder. Take both lua files that you downloaded
beforehand and put them into the folder scripts > bbsgl. Then launch LuaFrontend and activate both scripts.
Now you're good to go!

  • LuaBackend

You want to download LuaBackend and unpack the .zip folder. Take DGBHELP.dll and place it in
C:\Program Files\Epic Games\KH_1.5_2.5 or wherever you have KH 1.5 + 2.5 installed.
Now go to C:\Users\NAME\Documents\KINGDOM HEARTS HD 1.5+2.5 ReMIX\scripts\bbs and place both
lua files in here. If these folders don't exist yet you can go ahead and create them. Now you're good to go!

Both methods work with the randomizer, though most testing has been done with LuaBackend.
Keep in mind that this randomizer is a work in progress. Bugs will be fixed as they appear.
